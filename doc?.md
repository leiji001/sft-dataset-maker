# 🤖 小白也能学会的：AI 训练集制作神器使用手册

你好！欢迎使用本工具。无论你是小学生还是 80 岁的老奶奶，只要你会点鼠标、会打字，就能跟着这份教程，亲手做出属于你自己的 AI 训练集（SFT 数据集）。

我们把整个过程想象成**“把书里的知识喂给机器人吃”**。下面开始吧！

---

## 🛠 第一步：把“工具箱”搬回家（下载程序）

你现在看到的这些代码就像是一个零件箱，我们需要先把它下载到你的电脑里。

1. **如果你会用 Git**：直接运行 `git clone https://github.com/leiji001/sft-dataset-maker.git`。
2. **如果你不会 Git（推荐所有人）**：
   - 看到网页右上角那个绿色的 **「Code」** 按钮了吗？点它。
   - 在弹出的菜单里选择 **「Download ZIP」**。
   - 下载完成后，把这个压缩包“解压”（右键选择解压到当前文件夹）。
   - 记住这个文件夹的位置，下面我们要用到它。

---

## 🧹 第二步：请出“万能管家”（配置环境）

电脑需要一点特殊的“魔法”才能运行这个工具。我们需要安装一个叫 `Python` 的软件。

1. **下载 Python**：
   - 去这个网站：[python.org](https://www.python.org/downloads/)。
   - 点击那个大大的黄色按钮下载。
   - **重要：** 安装时一定要勾选底下的 **「Add Python to PATH」**（这就好比给管家一把开门的钥匙）。

2. **安装“加速工具” (uv)**：
   - 就像为了让管家跑得更快，我们需要给他一双跑鞋。
   - 打开电脑的“终端”（Windows 用户搜 `cmd` 或 `PowerShell`，Mac 用户搜 `终端/Terminal`）。
   - 输入并运行：`pip install uv`（这行拼写要完全正确哦）。

---

## 🔑 第三步：拿到 API“魔法钥匙”（配置模型）

这个工具需要请“大模型”（比如 DeepSeek 或 OpenAI）来帮忙写题目，所以我们需要给它一把“钥匙”。

1. **获取钥匙**：
   - 去你想使用的平台（推荐 DeepSeek：[platform.deepseek.com](https://platform.deepseek.com/)）。
   - 注册一个账号，找到 **「API Keys」**，点击 **「创建 API Key」**。
   - 把那一串长长的字母和数字给记下来（就像存银行卡密码一样，别给别人看）。

2. **告诉工具钥匙在哪**：
   - 在我们刚才下载的那个文件夹里，新建一个记事本文件。
   - 把名字改成 `.env`（没错，名字开头有一个点）。
     - *小贴士：如果 Windows 不让你改，你可以先打开记事本，点击“另存为”，文件名写 `".env"`（带双引号），这样就行了！*
   - 把下面这段话复制进去，并把 `你的魔法钥匙` 换成你刚才记下的那串钥匙：

   ```env
   LLM_PROVIDER=deepseek
   LLM_API_KEY=你的魔法钥匙
   LLM_BASE_URL=https://api.deepseek.com/v1
   ```

---

## 🌟 进阶小贴士：如果你有“带图片的复杂文档” (MinerU)

有些书里面有很多图片和奇怪的排版，普通管家可能看不懂。这时候我们可以请一位专业的**“高级翻译官” (MinerU)**。

1. **什么是 MinerU？** 这是一个专门用来识别复杂文档的服务。
2. **如何开启？** 
   - 你需要先运行一个 MinerU 服务（这通常需要另一台强大的电脑，或者你的电脑性能很好）。
   - 在你的 `.env` 文件里增加这两行：
     ```env
     # 告诉工具高级翻译官在哪里
     MINERU_API_URL=http://localhost:8888/file_parse
     MINERU_TIMEOUT=300
     ```
   - 这样在处理文档时，工具就会自动请这位翻译官来帮忙啦！

---

## 🚀 第四步：开始喂知识（运行程序）

现在，我们要把你的 PDF 或 Word 文档变成 AI 能读懂的“练习题”了。

1. **准备文档**：
   - 把你想让 AI 学习的文件（比如课本、说明书）放进一个文件夹里，起名叫 `docs`。
2. **下达命令**：
   - 回到刚才那个黑黑的命令行窗口。
   - 先进入我们的文件夹（比如输入 `cd sft-dataset-maker`）。
   - 输入并运行命令：
     ```bash
     uv run main.py ./docs/
     ```
   - 你会看到屏幕上开始冒出漂亮的表格和进度条，这说明机器人正在努力读取你的文档并出题呢！

---

## 🍎 第五步：收获你的“练习册”（查看结果）

运行结束后，去你的文件夹里找一个叫 **`output`** 的文件夹。

1. 打开它，你会发现一个叫 **`sft_dataset.jsonl`** 的文件。
2. 这个文件里装满了高质量的“问题”和“回答”，这就是你辛苦劳动的果实——**AI 训练集**！

---

### ❓ 遇到问题别担心：
- **画面报错了？** 检查一下 `.env` 文件里的钥匙是不是填错了。
- **文件不认识？** 确保你的文件是 PDF、Word (docx) 或者 PPT。
- **还是不懂？** 喝口热水，重新看一遍第一步。你最棒了，一定可以的！
